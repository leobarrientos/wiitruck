# wiitruck v1.0.0

Develop in python with POO.

## Hardware
* Raspberry Pi 3B+
* Puente H L298n
* Wii Mote (https://es.wikipedia.org/wiki/Wiimote)
* protoboard
* 4 leds
* Somes wires
